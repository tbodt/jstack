JStack

A reimplementation of a Java stack data structure.

Download this project as a .zip file Download this project as a tar.gz file

The Stack class in the Java library has some problems. It inherits from Vector, so it is very easy to insert elements into the middle of the stack:

stack.insertElementAt(new Object(), 0);

That would insert an Object at the bottom of the stack. Not the top, where elements are supposed to go.

The stack in JStack doesn't work like that. It has its own inheritance hierarchy of classes, designed the same way as the rest of the Java Collections Framework.
